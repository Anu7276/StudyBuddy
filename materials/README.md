# materials/ folder guide

This folder is where you upload the actual PDF files (notes, PYQs, MCQs, sample papers). It mirrors the site structure exactly: Class -> Subject -> Book/Section -> Chapter -> Resource type.

Drop a PDF into the matching chapter folder using a clear filename, e.g. `notes.pdf`, `pyq-2024.pdf`, `sample-paper-1.pdf`. Then open the matching chapter HTML page (in `/pages/classNN/subject/`) and paste the file's Google Drive share link in place of the `PASTE_PYQ_DRIVE_LINK_HERE` / `PASTE_SAMPLE_PAPER_DRIVE_LINK_HERE` placeholder.

Each chapter page also has an editable **Notes** section directly in its HTML — you don't need a PDF for notes specifically, you can paste text straight into the page (look for the `<!-- ADD MATERIAL -->` comment above the notes paragraph).
